#### 설치 후 업데이트

##### 업데이트 가능목록 확인 
```shell
sudo apt --upgradable
```

##### 리포지토리 업데이트
```shell
sudo apt update
```

##### 업데이트 실행
```shell
sudo apt upgrade
```

#### 필수 도구 설치

##### 네트워크 도구
```shell
sudo apt install net-tools
```

##### 기본 개발도구 설치
```shell
sudo apt install -y build-essential
```
